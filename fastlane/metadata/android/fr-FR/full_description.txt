# Forké avec des fonctionnalités supplémentaires

- Suppression de l'encombrement, comme les publicités et les liens.
- Vous pouvez renommer les applications dans le tiroir d'applications (le renommage des applications sur l'écran d'accueil est déjà pris en charge. Cliquez simplement longuement sur une application sur l'écran d'accueil et commencez à taper.)
- Beaucoup plus de possibilités de personnalisation des gestes de l'écran d'accueil
    - Les gestes sont maintenant :
        - Glissement vers le haut, vers le bas, vers la gauche, vers la droite
        - Possibilité de changer l'application qui s'ouvre en cliquant sur l'horloge
        - Possibilité de changer l'application qui s'ouvre en cliquant sur la date
    - Les actions possibles incluent maintenant :
        - Ouvrir l'application spécifiée
        - Afficher la liste des applications
        - Verrouillage de l'écran
        - Afficher les applications ouvertes récemment
        - Ouverture du panneau de notification
        - Ouverture des paramètres rapides
        - Boîte de dialogue d'alimentation
        - Prendre une capture d'écran
        - Désactiver le geste
- Vous pouvez également positionner l'horloge indépendamment des applications domestiques.
- Modifiez l'alignement des applications dans le tiroir d'applications indépendamment de l'écran d'accueil.
- Modifiez la taille de la police des sections de l'écran d'accueil indépendamment les unes des autres.
- Suppression de l'autorisation d'accès à Internet - Suppression de l'autorisation d'accès à Internet Vous ne savez jamais ce qu'un développeur d'applications veut savoir de vous.
- Verrouiller les paramètres derrière la biométrie (bascule)
- Ajout de la possibilité de signaler un crash par e-mail.
- Ajout de la possibilité d'afficher le temps d'utilisation de l'application.
